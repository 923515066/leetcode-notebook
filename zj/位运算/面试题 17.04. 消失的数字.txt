数组nums包含从0到n的所有整数，但其中缺了一个。请编写代码找出那个缺失的整数。你有办法在O(n)时间内完成吗？

注意：本题相对书上原题稍作改动

示例 1：

输入：[3,0,1]
输出：2
 

示例 2：

输入：[9,6,4,2,3,5,7,0,1]
输出：8


// 先从1..n异或一遍，然后在遍历异或数组中的数字，结果就为缺失的值
class Solution {
    public int missingNumber(int[] nums) {
        int ans = 0;
        for(int i = 1; i <= nums.length; i++) {
            ans ^= i;
        }
        for(int num : nums) {
            ans ^= num;
        }
        return ans;
    }
}
